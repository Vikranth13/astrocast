# AstroCast MVP

## Main Question

Is tonight a good night for stargazing?

## MVP Definition

The minimum viable product is a working web app where a user can enter a city and receive a stargazing score based on weather conditions.

## MVP User Flow

1. User enters a city.
2. App converts the city into latitude and longitude.
3. App fetches weather forecast data.
4. App calculates a stargazing score.
5. App displays the score and a short explanation.

## MVP Inputs

- City name

## MVP Outputs

- Location name
- Stargazing score from 0 to 100
- Rating such as Excellent, Good, Okay, Poor, or Bad
- Cloud cover
- Rain chance
- Wind speed
- Short explanation

## Out of Scope for MVP

- Login
- Database
- Saved user profiles
- Social sharing
- Mobile app
- AI chatbot
- Complex machine learning

# AstroCast

AstroCast is a stargazing forecast app that helps users decide whether tonight is good for observing the night sky.

## MVP Goal

The MVP allows a user to enter a location and receive a stargazing score based on weather and sky conditions.

## Problem

People interested in space often do not know whether the sky conditions are good enough for stargazing. AstroCast helps users quickly decide whether it is worth going outside to observe the night sky.

## Core User Story

As a user, I want to enter my city and see whether tonight is good for stargazing, so that I can decide whether to go outside.

## Planned Tech Stack

- Frontend: React, TypeScript, Vite
- Backend: Python, FastAPI
- APIs: Open-Meteo, NASA APOD
- Version Control: Git and GitHub

## MVP Features

- Search by city
- Convert city to latitude and longitude
- Fetch weather forecast data
- Calculate a stargazing score
- Show a simple explanation for the score
- Display astronomy-related content

## Non-MVP Features

These may be added later:
